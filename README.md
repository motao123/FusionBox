# FusionBox

一个功能丰富的 Linux 服务器管理脚本

## 功能模块

- **Proxy** - 基于 sing-box 的代理管理
- **System** - 系统信息、BBR、基准测试、备份恢复
- **Network** - IP查询、流媒体检测、网络测速、DNS测试
- **Web** - LNMP/LAMP一键部署、SSL证书、网站优化
- **Panels** - Docker管理、服务器面板、Aria2/Rclone
- **Market** - 60+应用一键安装

## 快速安装

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/motao123/FusionBox/main/install.sh)
```

## 使用方法

```bash
fusionbox proxy    # 代理管理
fusionbox system   # 系统管理
fusionbox network  # 网络工具
fusionbox web      # 网站部署
fusionbox panels   # 面板工具
fusionbox market   # 应用市场
```
