Features

核心特性

轻量、强大、易用 — FusionBox 让服务器管理变得简单

✓纯 Bash 实现，零外部依赖，任何 Linux 发行版开箱即用
✓支持 Debian / Ubuntu / CentOS / Fedora / Alpine 等主流发行版
✓支持 amd64 (x86_64) 和 arm64 (aarch64) 双架构
✓中文交互界面，交互菜单 + 命令行双模式自由切换
✓自动检测系统环境与依赖，智能提示，小白友好
✓完整的操作日志记录，方便审计与问题排查
✓内置一键更新机制，保持始终最新版本
✓MIT 开源协议，自由使用、修改和分发
